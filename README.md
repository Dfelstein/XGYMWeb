## Design System Compliance

All pages include:

### Consistent Navigation
- Sticky navigation with XGYM logo (gold "X" + "GYM")
- Links to Home, Testing, Training Library, Team, Coaching, Memberships
- "Book Now" CTA button (links to https://digitalpresence.biz/xgym/dexa-scan/#form)
- Mobile hamburger menu with smooth animations

### Consistent Footer
- 3-column layout with XGYM branding, quick links, and contact info
- Phone: 0424 023 601
- Email: admin@xgym.com.au
- Location: Castle Hill, NSW
- Copyright notice

### Color Palette
- Primary black: #0a0a0a
- Off-white text: #f5f2ec
- Gold accent: #c9a84c
- Light gold: #e8c96a
- Mid tone: #141414
- Grey text: #888

### Typography
- Headlines: Bebas Neue (0.08em letter-spacing)
- Body: Barlow (300/400/600 weights, 1.6 line-height)
- Labels/CTAs: Barlow Condensed (700/900 weights, 0.2em letter-spacing)
- Google Fonts: https://fonts.googleapis.com/css2?family=Bebas+Neue&family=Barlow:ital,wght@0,300;0,400;0,600;1,300&family=Barlow+Condensed:wght@700;900&display=swap

### Section Layout
- All sections use 8rem 3rem padding with max-width 1400px
- Section labels: gold uppercase text with 30px gold line before them
- Hero sections: 70vh height with dark gradient background (not 100vh for interior pages)
- Feature/card grids: responsive auto-fit layout with min-width constraints

### Interactive Elements
- Buttons: gold background (#c9a84c), black text, uppercase, hover to lighter gold
- Cards: border transitions and lift effect on hover
- Mobile responsive: hamburger menu activation and stacked layouts
